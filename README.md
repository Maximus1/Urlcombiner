# UrlCombiner

UrlCombiner ist eine einfache und nützliche Android-App, die entwickelt wurde, um das Kombinieren von URLs schnell und effizient zu gestalten. Speichern Sie eine oder mehrere Basis-URLs und kombinieren Sie diese anschließend über die "Teilen"-Funktion aus jeder beliebigen App heraus mit einer anderen URL.

## Funktionsweise

Die App löst ein einfaches Problem: Oft möchte man eine URL (z.B. von einem Nachrichtenartikel) schnell mit einer anderen URL (einem Web-Dienst) verknüpfen. Ein klassisches Beispiel ist das Archivieren einer Webseite mit einem Dienst wie `archive.is`.

Anstatt die URL zu kopieren, den Browser zu öffnen, die Archiv-Seite aufzurufen und die URL dort einzufügen, können Sie diesen Prozess mit UrlCombiner auf zwei Klicks reduzieren.

## Features

- **Basis-URLs verwalten:** Fügen Sie so viele Basis-URLs wie nötig hinzu, bearbeiten oder löschen Sie sie.
- **Nahtlose Integration:** Nutzen Sie die "Teilen"-Funktion in Ihrem Browser, auf YouTube, in News-Apps etc. und wählen Sie "UrlCombiner".
- **Intelligente Auswahl:**
    - Wenn Sie nur **eine** Basis-URL gespeichert haben, wird diese automatisch verwendet.
    - Bei **mehreren** Basis-URLs fragt die App, welche Sie verwenden möchten.
- **Einfach und schnell:** Kein Schnickschnack. Die App ist auf eine einzige Aufgabe optimiert und erledigt diese zuverlässig.
- **Mehrsprachig:** Die App unterstützt mehrere Sprachen.

## Wie man die App benutzt

1.  **Einrichtung:** Öffnen Sie die App und tippen Sie auf den **`+`** Button. Fügen Sie eine Basis-URL hinzu, die Sie häufig verwenden (z.B. `https://archive.is/` oder `https://12ft.io/`). Geben Sie ihr einen leicht wiedererkennbaren Namen.
2.  **Kombinieren:** Wenn Sie im Internet auf eine Seite stoßen, die Sie kombinieren möchten:
    - Öffnen Sie das "Teilen"-Menü Ihres Browsers.
    - Wählen Sie "UrlCombiner" aus der Liste der Apps.
    - Die kombinierte URL wird sofort in Ihrem Standard-Browser geöffnet. Fertig!
